## üöÄ Quick Start Guide

### Installation & Setup

```bash
npm install @supabase/supabase-js
```

```typescript
import { createClient } from "@supabase/supabase-js";
import { Database } from "./database.types"; // Generate with: supabase gen types typescript

const supabase = createClient<Database>("YOUR_SUPABASE_URL", "YOUR_SUPABASE_ANON_KEY");
```

### Essential Operations

```typescript
// üîç Fetch data
const { data, error } = await supabase.from("users").select("*").eq("active", true);

// ‚ûï Insert data
const { data, error } = await supabase
  .from("users")
  .insert({ name: "John", email: "john@example.com" })
  .select();

// üîê Sign up user
const { data, error } = await supabase.auth.signUp({
  email: "user@example.com",
  password: "secure-password",
});

// üì° Listen to changes
const channel = supabase
  .channel("users-channel")
  .on("postgres_changes", { event: "*", schema: "public", table: "users" }, (payload) =>
    console.log("Change received!", payload),
  )
  .subscribe();
```

### üõ°Ô∏è Row Level Security (RLS)

```sql
-- Always enable RLS on your tables
ALTER TABLE users ENABLE ROW LEVEL SECURITY;

-- Create policies for data access
CREATE POLICY "Users can view own data" ON users
  FOR SELECT USING (auth.uid() = id);
```

```typescript

const supabase = createClient(
  process.env.NEXT_PUBLIC_SUPABASE_URL!,
  process.env.NEXT_PUBLIC_SUPABASE_ANON_KEY!,
);

// ‚ùå Bad: Hardcoded keys

```

## üîß Common Patterns

### Pagination

```typescript
const ITEMS_PER_PAGE = 10;
const page = 1;

const { data, error, count } = await supabase
  .from("posts")
  .select("*", { count: "exact" })
  .range((page - 1) * ITEMS_PER_PAGE, page * ITEMS_PER_PAGE - 1)
  .order("created_at", { ascending: false });
```

### Error Handling

```typescript
const handleSupabaseOperation = async () => {
  try {
    const { data, error } = await supabase.from("users").select("*");

    if (error) {
      console.error("Supabase error:", error.message);
      return { success: false, error: error.message };
    }

    return { success: true, data };
  } catch (err) {
    console.error("Unexpected error:", err);
    return { success: false, error: "Unexpected error occurred" };
  }
};
```

### Conditional Queries

```typescript
let query = supabase.from("products").select("*");

if (category) query = query.eq("category", category);
if (minPrice) query = query.gte("price", minPrice);
if (searchTerm) query = query.ilike("name", `%${searchTerm}%`);

const { data, error } = await query;
```

## ‚ö° Performance Tips

### Query Optimization

```typescript
// ‚úÖ Good: Select only needed columns
const { data } = await supabase.from("users").select("id, name, avatar_url");

// ‚ùå Bad: Select all columns
const { data } = await supabase.from("users").select("*");
```

### Batch Operations

```typescript
// ‚úÖ Good: Batch insert
const { data, error } = await supabase.from("posts").insert([
  { title: "Post 1", content: "..." },
  { title: "Post 2", content: "..." },
]);

// ‚ùå Bad: Multiple single inserts
for (const post of posts) {
  await supabase.from("posts").insert(post);
}
```

### Connection Management

```typescript
// ‚úÖ Good: Reuse client instance
const supabase = createClient(url, key);

// ‚ùå Bad: Create new client for each operation
const getData = () => {
  const supabase = createClient(url, key); // Don't do this
  return supabase.from("table").select();
};
```

...options,

private \_listenForAuthEvents() {
let data = this.auth.onAuthStateChange((event, session) => {
this.\_handleTokenChanged(event, 'CLIENT', session?.access_token)
})
return data
}

private \_handleTokenChanged(
event: AuthChangeEvent,
source: 'CLIENT' | 'STORAGE',
token?: string
) {
if (
(event === 'TOKEN_REFRESHED' || event === 'SIGNED_IN') &&
this.changedAccessToken !== token
) {
this.changedAccessToken = token
} else if (event === 'SIGNED_OUT') {
this.realtime.setAuth()
if (source == 'STORAGE') this.auth.signOut()
this.changedAccessToken = undefined
}
}
